# Les Voix d'Ève — lesvoixdeve.ch

Site web de l'association **Les Voix d'Ève**, accompagnant les personnes victimes de violences conjugales en Suisse.

## Structure du projet

```
lesvoixdeve/
├── index.html          # Page d'accueil
├── prestations.html    # Page des prestations
├── css/
│   ├── style.css       # Styles page d'accueil
│   └── prestations.css # Styles page prestations
├── js/
│   ├── main.js         # JS page d'accueil (animations, nav)
│   └── nav.js          # JS navigation partagée
└── README.md
```

## Technologies

- HTML5 sémantique
- CSS3 (variables, grid, flexbox, animations keyframes)
- JavaScript vanilla (IntersectionObserver, scroll effects)
- Google Fonts : Cormorant Garamond + DM Sans

## Pages

### Page d'accueil (`index.html`)
- Navigation fixe avec effet scroll
- Section Hero avec animations d'entrée
- Section "Qui sommes-nous" avec cartes fondatrices
- Section Services (6 cartes)
- Section statistique choc
- Section "Notre approche" avec 4 piliers
- CTA final

### Page Prestations (`prestations.html`)
- Header de page
- Bloc cotisation / membership (CHF 50/an)
- Bloc accompagnement physique avec tableau des tarifs
- Bloc bénévolat
- Bloc CTA
- Note de bas de page
